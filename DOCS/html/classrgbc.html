◆ operator[]()

uint16_t & rgbc::operator[] ( int  index)

Indexing operator for rgbc value.

This operator[] allows for the capability of addressing the contents of an rgbc value by index in order R, G, B, C. Thus, to address the red, green, blue, and clear values, we can use rgbc[0], rgbc[1], rgbc[2], and rgbc[3] respectively. NOTE : operator[] does not check if the index you've entered is out of range because bmptk-make does not allow for throwing exceptions by default, instead it will return 0.

Returns
Reference to the addressed colour value, or 0 on index out of range
