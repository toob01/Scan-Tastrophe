 The \href{https://github.com/toob01/Scan-Tastrophe/}{\texttt{ Scan-\/\+Tastrophe}} is a rudimentary photoscanner with 4 x 4 pixel resolution, making use of the TCS3472 colour sensor. The codebase makes extensive use of \href{https://github.com/wovo/hwlib}{\texttt{ hwlib}} to address an Arduino Due microcontroller. With use of \href{https://github.com/wovo/bmptk}{\texttt{ bmptk}} it\textquotesingle{}s possible to compile the program for other microcontrollers as well.

The codebase is documented with Doxygen comments, and generated documentation can be found in the DOCS folder.\hypertarget{index_autotoc_md1}{}\doxysection{Repository Contents}\label{index_autotoc_md1}
Scan-\/\+Tastrophe includes one library for use with the TCS34725/7 range of colour sensors in combination with \href{https://github.com/wovo/hwlib}{\texttt{ hwlib}}.

Additionally it includes two Abstract datatypes that the TCS34725/7 library makes use of\+:
\begin{DoxyItemize}
\item RGBC (red, green, blue, clear) stored in 16-\/bit unsigned integers
\item RGB (red, green, blue) stored in 8-\/bit unsigned integers
\end{DoxyItemize}

Both ADTs are provided with tests using the \href{https://github.com/catchorg/Catch2}{\texttt{ Catch2 testing framework}} \begin{quote}

\hypertarget{index_autotoc_md2}{}\doxysection{Features}\label{index_autotoc_md2}
\hypertarget{index_autotoc_md3}{}\doxysubsection{TCS34725/7 library}\label{index_autotoc_md3}
The TCS34725/7 library provides the following functionality\+:
\begin{DoxyItemize}
\item Configuration Macros for registers and their values
\item Reading from specific registers
\item Reading colour data from corresponding registers
\item Explicitly put device in sleep and idle state
\item Explicitely starting an RGBC cycle
\item Reading and returning all colour data
\item Converting raw colour data to 8-\/bit RGB values or 24-\/bit integer (hex) value
\end{DoxyItemize}\hypertarget{index_autotoc_md4}{}\doxysubsection{RGBC and RGB ADTs}\label{index_autotoc_md4}
The RGBC and RGB abstract datatypes are to store the data one might receive from the TCS3472x colour sensor RGBC stores four 16-\/bit unsigned integers, one for each of the Red, Green, Blue, and Clear colour channels. RGB stores three 8-\/bit unsigned integers, one for each 8-\/bit representation of the values as described in the RGBC datatype.

Both provide a range of operators to make it easy to interact with and modify colour data.

Operators included are\+: +, +=, -\/, -\/=, $\ast$, $\ast$=, /, /=, $\vert$, $\vert$=, \&, \&=, ==, !=, \texorpdfstring{$<$}{<}, \texorpdfstring{$<$}{<}=, \texorpdfstring{$>$}{>}, \texorpdfstring{$>$}{>}=, \mbox{[}\mbox{]}, \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<}
